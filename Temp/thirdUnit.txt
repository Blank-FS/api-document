## 三方提供的接口

### 1.商品审核通知接口

```
说明：推送到平台的商品，需要经过审核才会上架展示。平台审核后，我们会将审核结果通知给贵方，包含审核结果，审核拒绝原因等。
```



### 2.获取预约排期的接口

```
说明：获取套餐实时排期的接口，用于在商品详情页和预约页面展示。另外，我们可能会定期查询此接口以获取套餐排期状态，用来过滤无排期的套餐。
```



### 3.提交预约接口

```
说明：用户在160平台支付后，会进行预约，此时我们会将用户预约信息同步给贵方，需要返回贵方的订单id。
```



### 4.预约改期接口

```
说明：给用户提供修改预约的能力
```



### 5. 取消预约接口

```
说明：提供用户退款能力
```
