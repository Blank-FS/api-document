## 电商项目简介

### 简介
健康160是互联网医疗健康服务平台，提供从体检、挂号、问诊到健康商城、健康管理等一站式医疗大健康服务。健康商城开放平台帮助商家（或代理商）更高效的运营，开放商品管理、订单管理、售后服务等能力，商家（代理商）可以通过平台开放的 API 接口与商家内部系统互通互联。

### 入驻介绍
新商家可以注册成《160云医院》用户，访问 https://h.91160.com/vue/home 进行注册

### 电商开放能力

| 模块 | 接口 | 
| ------- | ------- |
|   商品      |    商品流程介绍<br>创建商品<br>修改商品信息<br>商品上下架<br>商品详情<br>商品列表<br>删除商品    |
|   订单      |    订单流程介绍<br>订单列表查询<br>订单详情<br>订单统计<br>订单核销    |
|   售后      |    售后流程介绍<br>退款列表<br>退款审核    |
|   预约排班      |    预约排班流程介绍<br>创建预约模板<br>预约模板列表<br>编辑预约模板<br>删除预约模板<br>预约模板关联商品   |
